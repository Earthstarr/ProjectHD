# ProjectHD

UE5.5 TPS 게임입니다.

## 개요

압도적 화력을 쏟아붓는 스트라타젬 시스템을 이용하여 미션을 수행하고 몰려오는 적들을 섬멸하는 전술적 액션 게임입니다.

## 기본 기능

- 캐릭터 이동
- 무기 교체
- 발사
- 탄약 / 재장전
- 체력 시스템 + HUD
- 스트라타젬(Stratagem) 시스템 : Q를 눌러 입력

## 구현 예정 기능

- 몬스터 구현
- 몬스터 AI 구현
- 미션 구현
- 시작 컷씬
- 엔딩 컷씬

세부 구현 계획은 [Issues](../../issues)에서 확인할 수 있습니다.

## 브랜치 전략

| 브랜치 | 용도 |
|--------|------|
| main | 안정된 버전 |
| feature/* | 기능 개발 |

## 커밋 컨벤션

```
feat: 새로운 기능 추가
fix: 버그 수정
asset: 에셋 추가/수정
refactor: 코드 정리
docs: 문서 수정
chore: 설정 파일 등 기타 수정
```
